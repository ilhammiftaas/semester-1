# FINAL PROJECT SEMESTER - 1

Program kasir ini menggunakan bahasa java dengan berbagai keunggulan di setiap role yang tersedia
  1. Kasir (Pin: 1111)
     Fokus dari role ini adalah kecepatan dan akurasi transaksi dengan keunggulan:
       - Sistem keranjang fleksibel
       - Validasi pembayaran
       - Perhitungan diskon otomatis
       - Penggabungan kuantitas otomatis
  2. Admin (Pin: 2222)
     Fokus dari role ini adalah manajemen master data (Menu) dengan keunggulan:
       - Sistem CRUD (Create Read Update Delate)
       - Update fleksibel (Kosongi jika tidak diubah)
       - Pencarian pintar (Partial Search)
  3. Owner (Pin: 3333)
     Fokus dari role ini adalah pelaporan, analisa bisnis (Business Intelligence), dan pengambilan keputusan dengan keunggulan:
       - Laporan finansial komprehensif
       - Analisa produk terlaris (Top Selling)
       - Laporan harian otomatis
